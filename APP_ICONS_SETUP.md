## React Native WebView Integration

To use this game in React Native WebView, you have 3 options:

### Option 1: Use Deployed URL (Recommended for Production)
```jsx
import { WebView } from 'react-native-webview';

<WebView 
  source={{ uri: 'https://your-app.vercel.app' }}
  style={{ flex: 1 }}
/>
```

### Option 2: Local Development Server
```jsx
// Start dev server: cd web && npm run dev
// Find your IP: ifconfig (Mac) or ipconfig (Windows)
<WebView 
  source={{ uri: 'http://192.168.1.X:5173' }}
  style={{ flex: 1 }}
/>
```

### Option 3: Bundle HTML Locally (Best for Offline)
```jsx
<WebView 
  source={require('./web/dist/index.html')}
  originWhitelist={['*']}
  style={{ flex: 1 }}
/>
```
